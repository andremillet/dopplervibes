# DopplerVibes

## Ideia

Um dispositivo que recebe entradas de uma máquina de ultrassom Doppler capturando batimentos cardíacos fetais, transforma o áudio em vibrações que podem ser sentidas com as mãos, direcionado a gestantes com deficiências auditivas.

## Plano

- Pesquisar como capturar entrada de áudio de dispositivos Doppler de ultrassom (por exemplo, via conector de áudio, Bluetooth ou API direta)
- Analisar o som de batimentos cardíacos Doppler para extrair ritmo e padrões para conversão em vibração
- Projetar sistema de feedback háptico (por exemplo, motores de vibração, atuadores) para simular batimentos cardíacos sentidos
- Prototipar formato do dispositivo adequado para gestantes (por exemplo, almofada ou luva wearable)
- Garantir segurança e usabilidade para usuárias gestantes com deficiência auditiva, incluindo aprovações médicas
